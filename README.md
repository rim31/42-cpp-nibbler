## 42-cpp-nibbler
C++ : snake within 3 library graphics
SDL / SFML / NCURSES

play and change the library graphic instantanely (information about its forbidden)

## SNAKE : game in 3 dynamic libraries

# RUN
```C++
make
./run 42 42 sdl
```
# play
WASD (qwerty keybord)
pause : P
colors
switch library : 1 2 3 (above f1 f2 f3)
speed increasing
ESC to quit
